1пјҢеҝ«жҚ·й”®жҹҘйҳ…

гҖҗ Ctrl+Shift+L гҖ‘ В {Help->Default Keymap Reference}

Show key assist В  жү“ејҖеҝ«жҚ·й”®жҹҘйҳ…зӘ—еҸЈ

2пјҢй…ҚзҪ® еҝ«жҚ·й”®

еңЁ[Window -> Preferences -> General -> keys] {File->Settings->Keymap}дёӢпјҢеҸҜеңЁжӯӨз•Ңйқўдҝ®ж”№еҝ«жҚ·й”®гҖӮ

****3пјҢзј–иҫ‘зӣёе…іеҝ«жҚ·й”® ****

3.1. гҖҗAlt+/гҖ‘ Content Assist В В жҸҗдҫӣеҶ…е®№иҫ…еҠ©еҠҹиғҪпјҢеҪ“иҫ“е…ҘйғЁеҲҶзұ»гҖҒеұһжҖ§жҲ–ж–№жі•зҡ„еҗҚеӯ—еҗҺпјҢжҢүжӯӨй”®дјҡжҳҫзӨәеҢ№й…Қзҡ„еҗҚз§°гҖӮ

Intellij Ideaй»ҳи®ӨжҳҜеҢәеҲҶеӨ§е°ҸеҶҷзҡ„пјҢеҸҜеңЁ{Settings->Editor->Code Completion}дёӯе°ҶCase sensitive completionи®ҫзҪ®дёәNoneгҖӮ

{Ctrl+Space} Basic code completion (the name of any class, methodВ  or variable)

{Ctrl+Shift+Space} Smart code completion (filters the list of methods andВ  variables by expected type)

{Ctrl+Alt+Space} Class name completion (the name of any project class independently of current imports)

{Ctrl+Shift+Enter} Complete statement

{Alt+Insert} Generate code… (Getters, Setters, Constructors,В  hashCode/equals, toString)

{Ctrl+O} Override methods

{Ctrl+I}В  Implement methods

{Tab} иҫ“е…ҘйғЁеҲҶд»Јз ҒеҗҺиҮӘеҠЁз”ҹжҲҗ

{Ctrl+J} еј№еҮәеҜ№иҜқжЎҶпјҢеҸҜйҖүжӢ©з”ҹжҲҗmainеҮҪж•°зӯүгҖӮиҫ“е…Ҙpsvm+tabпјҢеҲҷдјҡдә§з”ҹmainж–№жі•пјӣиҫ“е…Ҙiter+tabпјҢеҲҷз”ҹжҲҗforeachиҜӯеҸҘгҖӮ

3.2. гҖҗCtrl+/ гҖ‘гҖҗCtrl+Shift+C гҖ‘{Ctrl+/} В В еҝ«йҖҹж·»еҠ жіЁйҮҠпјҢиғҪдёәеҪ“еүҚиЎҢжҲ–йҖүе®ҡиЎҢеҝ«йҖҹж·»еҠ жіЁйҮҠжҲ–еҸ–ж¶ҲжіЁйҮҠгҖӮгҖҗ Ctrl+Shift+/ гҖ‘{Ctrl+Shift+/ } ж·»еҠ жіЁйҮҠеқ—гҖӮ

3.3. гҖҗCtrl+D гҖ‘{Ctrl+Y} В еҲ йҷӨеҪ“еүҚиЎҢжҲ–йҖүе®ҡиЎҢгҖӮ

3.4.гҖҗCtrl+Shift+O гҖ‘ В еҝ«йҖҹз”ҹжҲҗimport пјҢеҜје…ҘйңҖиҰҒзҡ„зұ»гҖӮВ {Alt+Enter}еңЁзұ»еҗҚеҗҺжҢүжӯӨз»„еҗҲй”®еҜје…Ҙзұ»гҖӮ{Ctrl+Alt+O}дјҳеҢ–еҜје…ҘгҖӮ 3.5. гҖҗCtrl+Shift+F гҖ‘{Ctrl+Shift+L} В ж јејҸеҢ–д»Јз ҒпјҢйҖүе®ҡеҗҺжҢүгҖҗCtrl+Shift+F гҖ‘еҝ«жҚ·й”®еҸҜд»Ҙж јејҸеҢ–иҝҷж®өд»Јз ҒпјҢеҰӮжһңдёҚйҖүе®ҡд»Јз ҒеҲҷй»ҳи®Өж јејҸеҢ–еҪ“еүҚж–Үд»¶гҖӮ 3.6. гҖҗ Ctr+Q гҖ‘{Ctrl+Shift+Backspace} В иҝ”еӣһеҲ°дёҠж¬Ўзј–иҫ‘зҡ„ең°ж–№гҖӮ

3.7.гҖҗTab гҖ‘{Tab} йҖүдёӯйғЁеҲҶеҗ‘еҸіи·ізҪ®tab зҡ„и·қзҰ» В гҖҗ Shift + Tab гҖ‘{Shift+Tab}йҖүдёӯйғЁеҲҶ еҗ‘е·Ұи·ізҪ®tab зҡ„и·қзҰ»

3.8.гҖҗShift +йј ж Үе·Ұй”®гҖ‘{Shift +йј ж Үе·Ұй”®} В д»ҺеҺҹе…үж ҮдҪҚзҪ®еҲ°еҪ“еүҚе…үж ҮдҪҚзҪ®йҖүдёӯеҢәеҹҹгҖӮ

3.9.гҖҗAlt+Shift+RгҖ‘{Shift+F6} В Rename йҮҚе‘ҪеҗҚйҖүдёӯзҡ„еҸҳйҮҸжҲ–ж–№жі•

3.10.гҖҗAtl+Shift+CгҖ‘{Ctrl+F6} В Change method signatureВ  еҜ№йҖүдёӯзҡ„ж–№жі•иҝӣиЎҢйҮҚжһ„пјҢдҝ®ж”№ж–№жі•еҗҚз§°гҖҒеҸӮж•°зӯүгҖӮ 3.11. гҖҗ Atl+Shift+L гҖ‘{Ctrl+Alt+V} В жҸҗеҸ–жң¬ең°еҸҳйҮҸ 3.12. гҖҗ Atl+Shift+M гҖ‘{Ctrl+Alt+M} В жҸҗеҸ–ж–№жі• 3.13.гҖҗ Atl+ вҶ‘гҖ‘ гҖҗ Atl+вҶ“ гҖ‘{Ctrl+Shift+вҶ‘} {Alt+Shift+вҶ‘} {Ctrl+Shift+вҶ“ } {Alt+Shift+вҶ“ } В е°ҶеҪ“еүҚиЎҢжҲ–йҖүе®ҡзҡ„д»Јз ҒдёҠз§»/дёӢз§»

3.14. гҖҗAlt+Shit+A гҖ‘{Alt+Shift+Insert} В еҲҮжҚўдёәеқ—йҖүжӢ©жЁЎејҸпјҢиҝҷж—¶е…үж ҮдјҡеҸҳдёәвҖң+вҖқ пјҢеҸҜжӢ–еҠЁйј ж ҮйҖүдёӯжҹҗдёҖеҢәеҹҹпјҢеҲ йҷӨдёҖеҲ—жҲ–еӨҡеҲ—д»Јз Ғж—¶еҫҲжңүз”ЁгҖӮ 3.15. гҖҗCtrl+M гҖ‘ {Ctrl+Shift+F12} В еҪ“еүҚи§ҶеӣҫпјҲViewпјүжңҖеӨ§еҢ–е’ҢиҝҳеҺҹ 3.16.гҖҗCtrl+Shift+Z гҖ‘ гҖҗCtrl+Shift+YгҖ‘{Ctrl+Shift+U} В  еӨ§е°ҸеҶҷиҪ¬жҚў

4пјҢжҹҘзңӢе’Ңе®ҡдҪҚеҝ«жҚ·й”®

4.1.гҖҗCtrl+O гҖ‘{Ctrl+F12} В жҳҫзӨәзұ»дёӯж–№жі•е’ҢеұһжҖ§зҡ„еӨ§зәІпјҢиғҪеҝ«йҖҹе®ҡдҪҚзұ»зҡ„ж–№жі•е’ҢеұһжҖ§гҖӮ 4.2. гҖҗCtrl+K гҖ‘гҖҒгҖҗCtrl+Shift+K гҖ‘В еҝ«йҖҹеҗ‘дёӢе’Ңеҗ‘дёҠжҹҘжүҫйҖүе®ҡзҡ„еҶ…е®№гҖӮ 4.3. гҖҗCtrl+Shift+T гҖ‘ {Ctrl+N} В жҹҘжүҫе·ҘдҪңз©әй—ҙдёӯзҡ„Java ж–Үд»¶пјҢеҸҜд»ҘдҪҝз”ЁвҖң*вҖқ гҖҒвҖң пјҹвҖқ зӯүйҖҡй…Қз¬ҰгҖӮ 4.4. гҖҗCtrl+Shift+R гҖ‘ {Ctrl+Shift+N}В е’ҢгҖҗCtrl+Shift+T гҖ‘еҜ№еә”пјҢжҹҘжүҫе·ҘдҪңз©әй—ҙдёӯзҡ„жүҖжңүж–Үд»¶пјҲеҢ…жӢ¬Java ж–Үд»¶пјүпјҢеҸҜд»ҘдҪҝз”ЁйҖҡй…Қз¬ҰгҖӮ 4.5. гҖҗCtrl+Shift+G гҖ‘ {Alt+F7}(жҲ–з”ЁеҸій”®иҸңеҚ•)В жҹҘжүҫзұ»гҖҒж–№жі•е’ҢеұһжҖ§зҡ„еј•з”ЁгҖӮеҰӮиҰҒйҮҚе‘ҪеҗҚжҹҗдёӘж–№жі•пјҢеҸҜд»ҘйҖҡиҝҮгҖҗCtrl+Shift+G гҖ‘еҝ«жҚ·й”®иҝ…йҖҹе®ҡдҪҚеҲ°жүҖжңүеј•з”ЁжӯӨж–№жі•зҡ„д»Јз ҒгҖӮ

4.6.гҖҗF3 гҖ‘гҖҗCtr пјӢеҚ•еҮ» гҖ‘{Ctr пјӢеҚ•еҮ»} В еҝ«йҖҹе®ҡдҪҚе…үж ҮдҪҚзҪ®зҡ„жҹҗдёӘзұ»гҖҒж–№жі•е’ҢеұһжҖ§гҖӮ 4.7. гҖҗAlt+вҶҗ гҖ‘гҖҒгҖҗAlt+вҶ’ гҖ‘ {Ctrl+Alt+вҶҗ } {Ctrl+Alt+вҶ’ } В еҗҺйҖҖеҺҶеҸІи®°еҪ•е’ҢеүҚиҝӣеҺҶеҸІи®°еҪ•пјҢеңЁи·ҹиёӘд»Јз Ғж—¶йқһеёёжңүз”ЁпјҢеҪ“жҹҘжүҫдәҶеҮ дёӘжңүе…іиҒ”зҡ„ең°ж–№пјҢдҪҶеҸҜиғҪи®°дёҚжё…жҘҡдәҶпјҢеҸҜд»ҘйҖҡиҝҮиҝҷдёӨдёӘеҝ«жҚ·й”®е®ҡдҪҚжҹҘжүҫзҡ„йЎәеәҸгҖӮВ {Ctrl+E} жңҖиҝ‘зј–иҫ‘зҡ„ж–Үд»¶

4.8.гҖҗCtrl+L гҖ‘{Ctrl+G} В е®ҡдҪҚеҲ°еҪ“еүҚзј–иҫ‘еҷЁзҡ„жҹҗдёҖиЎҢпјҢеҜ№йқһJava ж–Үд»¶д№ҹжңүж•ҲгҖӮ

4.9.гҖҗCtr+HгҖ‘{Ctrl+Shift+F} В жү“ејҖжҹҘжүҫеҜ№иҜқжЎҶпјҢ еҸҜд»ҘжҹҘжүҫеҪ“еүҚйҖүе®ҡзҡ„зұ»гҖҒж–№жі•зӯүгҖӮ 4.10. гҖҗ Ctr+Shift+E гҖ‘В еңЁеҚ•зӢ¬зҡ„зӘ—еҸЈдёӯпјҢеҲ—еҮәеҪ“еүҚжү“ејҖзҡ„ жүҖжңү ж–Үд»¶пјҢеҸҜд»Ҙе…ій—ӯйҖүе®ҡзҡ„ж–Үд»¶жҲ–еҲҮжҚўеҲ°жҹҗдёҖж–Үд»¶гҖӮВ гҖҗ Ctr+E гҖ‘еңЁQuick Swtich EditorзӘ—еҸЈдёӯеҲ—еҮәеҪ“еүҚжү“ејҖзҡ„ жүҖжңү ж–Үд»¶пјҢеҲҮжҚўж–Үд»¶жӣҙж–№дҫҝгҖӮ

4.11.гҖҗCtr+TгҖ‘{Ctrl+H}

5пјҢQuick Hierarchy жҹҘзңӢзұ»еұӮж¬Ўе…ізі»гҖӮ

гҖҗF4 гҖ‘ жҳҫзӨәзұ»зҡ„з»§жүҝе…ізі»пјҢе№¶жү“ејҖзұ»з»§жүҝи§ҶеӣҫгҖӮ ****и°ғиҜ•еҝ«жҚ·й”® **** 1. гҖҗCtrl+Shift+B гҖ‘{Ctrl+F8}пјҡеңЁеҪ“еүҚиЎҢи®ҫзҪ®ж–ӯзӮ№жҲ–еҸ–ж¶Ҳи®ҫзҪ®зҡ„ж–ӯзӮ№гҖӮ 2. гҖҗF11 гҖ‘{Shift+F9}пјҡи°ғиҜ•жңҖеҗҺдёҖж¬Ўжү§иЎҢзҡ„зЁӢеәҸгҖӮ 3. гҖҗCtrl+F11 гҖ‘{Shift+F10}пјҡиҝҗиЎҢжңҖеҗҺдёҖж¬Ўжү§иЎҢзҡ„зЁӢеәҸгҖӮ 4. гҖҗF5 гҖ‘{F7}пјҡи·ҹиёӘеҲ°ж–№жі•дёӯгҖӮеҪ“зЁӢеәҸжү§иЎҢеҲ°жҹҗж–№жі•ж—¶пјҢеҸҜд»ҘжҢүгҖҗF5 гҖ‘й”®и·ҹиёӘеҲ°ж–№жі•дёӯгҖӮ 5. гҖҗF6 гҖ‘{F8}пјҡеҚ•жӯҘжү§иЎҢзЁӢеәҸгҖӮ 6. гҖҗF7 гҖ‘пјҡжү§иЎҢе®Ңж–№жі•пјҢиҝ”еӣһеҲ°и°ғз”ЁжӯӨж–№жі•зҡ„еҗҺдёҖжқЎиҜӯеҸҘгҖӮ 7. гҖҗF8 гҖ‘пјҡз»§з»ӯжү§иЎҢпјҢеҲ°дёӢдёҖдёӘж–ӯзӮ№жҲ–зЁӢеәҸз»“жқҹгҖӮ

****еёёз”Ёзј–иҫ‘еҷЁеҝ«жҚ·й”® **** 1. гҖҗCtrl+C гҖ‘{Ctrl+C} {Ctrl+Insert}пјҡеӨҚеҲ¶(IntellijеҸҜеӨҚеҲ¶еҪ“еүҚиЎҢ)гҖӮ 2. гҖҗCtrl+X гҖ‘{Ctrl+X} {Shift+Delete}пјҡеүӘеҲҮ(IntellijеҸҜеүӘеҲҮеҪ“еүҚиЎҢпјүгҖӮ 3. гҖҗCtrl+V гҖ‘{Ctrl+V} {Shift+Insert}пјҡзІҳиҙҙгҖӮ{Ctrl+Shift+V} еј№еҮәеҜ№иҜқжЎҶпјҢйҖүжӢ©зІҳиҙҙгҖӮ 4. гҖҗCtrl+S гҖ‘{Ctrl+S }пјҡдҝқеӯҳж–Үд»¶гҖӮ 5. гҖҗCtrl+Z гҖ‘{Ctrl+Z}пјҡж’Өй”ҖгҖӮ 6. гҖҗCtrl+Y гҖ‘{Ctrl+Shift+Z}пјҡйҮҚеӨҚгҖӮ 7. гҖҗCtrl+F гҖ‘{Ctrl+F}пјҡжҹҘжүҫгҖӮ 8. гҖҗCtrl+Alt+вҶ‘ гҖ‘{Ctrl+D}: еӨҚеҲ¶ж•ҙиЎҢжҲ–еқ—гҖӮ ****е…¶д»–еҝ«жҚ·й”® **** 1. гҖҗCtrl+F6 гҖ‘{Alt+вҶ’}пјҡеҲҮжҚўеҲ°дёӢдёҖдёӘзј–иҫ‘еҷЁгҖӮ 2. гҖҗCtrl+Shift+F6 гҖ‘{Alt+вҶҗ }пјҡеҲҮжҚўеҲ°дёҠдёҖдёӘзј–иҫ‘еҷЁгҖӮ 3. гҖҗCtrl+F7 гҖ‘пјҡеҲҮжҚўеҲ°дёӢдёҖдёӘи§ҶеӣҫгҖӮ 4. гҖҗCtrl+Shift+F7 гҖ‘пјҡеҲҮжҚўеҲ°дёҠдёҖдёӘи§ҶеӣҫгҖӮ 5. гҖҗCtrl+F8 гҖ‘пјҡеҲҮжҚўеҲ°дёӢдёҖдёӘйҖҸи§ҶеӣҫгҖӮ 6. гҖҗCtrl+Shift+F8 гҖ‘пјҡеҲҮжҚўеҲ°дёҠдёҖдёӘйҖҸи§ҶеӣҫгҖӮ
