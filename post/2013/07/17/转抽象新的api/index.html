TabCompat.zip

еҒҮи®ҫжӮЁжғіеңЁзЁӢеәҸдёӯдҪҝз”Ё[action bar tabs](http://developer.android.com/guide/topics/ui/actionbar.html#Tabs)дҪңдёәдё»иҰҒзҡ„еҜјиҲӘж–№ејҸгҖӮ дёҚе№ёзҡ„жҳҜпјҢActionBar APIеҸӘжңүеңЁ3.0+зүҲжң¬дёӯжүҚиғҪдҪҝз”ЁгҖӮжүҖд»ҘеҰӮжһңжӮЁжғіи®©жӮЁзҡ„еә”з”Ёд№ҹиғҪиҝҗиЎҢеҲ°ж—§зҡ„и®ҫеӨҮдёӯпјҢжӮЁйңҖиҰҒжғіеҠһжі•ж”ҜжҢҒж—§зҡ„зі»з»ҹгҖӮ

еңЁжң¬зӨәдҫӢдёӯпјҢжӮЁе°ҶеҲӣе»әдёҖдёӘtab UIжҺ§д»¶жҺҘеҸЈпјҢиҜҘжҺҘеҸЈеңЁдёҚеҗҢзҡ„AndroidзүҲжң¬дёӯдҪҝз”ЁдёҚеҗҢзҡ„е®һзҺ°ж–№ејҸгҖӮжң¬иҜҫзЁӢд»Ӣз»ҚеҰӮдҪ•еҲӣе»әиҝҷж ·дёҖдёӘжҠҪиұЎжҺҘеҸЈгҖӮ

з ”з©¶жҠҪиұЎжҺҘеҸЈ

Abstraction В еңЁJavaиҜӯиЁҖдёӯж„Ҹе‘ізқҖйҡҗи—Ҹе®һзҺ°з»ҶиҠӮгҖӮеңЁдҪҝз”Ёж–°зҡ„APIж—¶пјҢеҸҜд»ҘдҪҝз”ЁжҠҪиұЎжңәеҲ¶жқҘ еҲӣе»әдёҖдёӘзүҲжң¬зӣёе…ізҡ„жҺ§д»¶пјҢиҜҘжҺ§д»¶еңЁж–°зүҲжң¬дёӯдҪҝз”Ёж–°APIпјҢеңЁж—§зүҲжң¬дёӯдҪҝз”Ёж—§APIгҖӮ

еҪ“дҪҝз”Ёиҝҷз§Қж–№ејҸзҡ„ж—¶еҖҷпјҢжӮЁйңҖиҰҒзЎ®е®ҡеңЁж—§зүҲжң¬дёӯйңҖиҰҒдҪҝз”ЁйӮЈдәӣж–°зүҲжң¬зҡ„зү№жҖ§пјҢз„¶еҗҺеҹәдәҺж–°зҡ„жҺҘеҸЈжқҘеҲӣе»әдёҖдёӘжҠҪиұЎзұ»гҖӮеңЁе®ҡд№үж–°зҡ„жҠҪиұЎжҺҘеҸЈзҡ„ж—¶еҖҷпјҢеә”иҜҘе°ҪеҸҜиғҪзҡ„еҸҚжҳ еҮәж–°зҡ„зү№жҖ§пјҲе°ҪеҸҜиғҪзҡ„еӨҚеҲ¶ж–°зҡ„APIе®ҡд№үпјүгҖӮиҝҷж ·еҸҜд»Ҙе°ҪеҸҜиғҪзҡ„еҗ‘еүҚе…је®№пјҢе№¶дё”еҪ“жҹҗдёҖеӨ©жӮЁи§үеҫ—дёҚеҶҚйңҖиҰҒжҠҪиұЎеұӮзҡ„ж—¶еҖҷеҫҲе®№жҳ“зҡ„еҲҮжҚўеӣһжқҘгҖӮ

дёҖж—Ұдёәж–°зҡ„APIеҲӣе»әдәҶжҠҪиұЎжҺҘеҸЈеҗҺпјҢе°ұеҸҜд»Ҙе®ҡд№үеӨҡдёӘе®һзҺ°зұ»е№¶дё”еңЁиҝҗиЎҢж—¶жқҘйҖүжӢ©дҪҝз”ЁйӮЈдёӘе®һзҺ°ж–№ејҸгҖӮдёҚеҗҢзҡ„API levelеҜ№еә”зҡ„е®һзҺ°ж–№ејҸеҸҜиғҪдёҚдёҖж ·гҖӮ

еҲӣе»әжҠҪиұЎеҗҺзҡ„TabжҺҘеҸЈ

дёәдәҶеҲӣе»әеҗ‘еҗҺе…је®№зүҲжң¬зҡ„tabsпјҢйңҖиҰҒйҰ–е…Ҳе®ҡд№үжӮЁзҡ„зЁӢеәҸйңҖиҰҒе“Әдәӣзү№жҖ§е’Ңе…·дҪ“зҡ„APIгҖӮеңЁиҝҷйҮҢзҡ„йЎ¶йғЁеҜјиҲӘtabдёӯпјҢжҲ‘д»¬ еҒҮи®ҫжңүеҰӮдёӢзү№жҖ§пјҡ

  1. Tab жҢҮзӨәеҷЁпјҢеҢ…еҗ«ж–Үеӯ—е’ҢеӣҫзүҮ
  2. Tabs еҸҜд»Ҙе’ҢFragmentе…іиҒ”иө·жқҘ
  3. Activity еә”иҜҘеҸҜд»Ҙзӣ‘еҗ¬еҲ°TabеҲҮжҚўдәӢд»¶ йў„е…ҲеҮҶеӨҮиҝҷдәӣйңҖжұӮеҸҜд»Ҙи®©дҪ зІҫзЎ®зҡ„жҺ§еҲ¶жҠҪиұЎеұӮзҡ„иҢғеӣҙгҖӮиҝҷж„Ҹе‘ізқҖжӮЁеҸҜд»ҘдҪҝз”Ёиҫғе°‘зҡ„ж—¶й—ҙеҲӣе»әеӨҡдёӘжҠҪиұЎеұӮе®һзҺ°пјҢе№¶дё”е°ҪеҸҜиғҪж—©зҡ„дҪҝз”Ёе…је®№жҖ§зҡ„е®һзҺ°гҖӮ

иҝҷйҮҢз”ЁдәҺTabsзҡ„е…ій”®APIдёәВ [ActionBar](http://developer.android.com/reference/android/app/ActionBar.html)В е’Ң [ActionBar.Tab](http://developer.android.com/reference/android/app/ActionBar.Tab.html)гҖӮ жӮЁйңҖиҰҒжҠҪиұЎиҝҷдәӣAPIжқҘдҪҝжӮЁзҡ„Tabsе’Ңзі»з»ҹзүҲжң¬е…іиҒ”иө·жқҘгҖӮиҝҷдёӘйЎ№зӣ®дёӯйңҖиҰҒж”ҜжҢҒEclair (API level 5) еҗҢж—¶иғҪеӨҹдҪҝз”ЁHoneycomb (API Level 11)зүҲжң¬дёӯзҡ„ж–°зү№жҖ§гҖӮдёӢеӣҫжҳҫзӨәдәҶиҜҘжҠҪиұЎAPIеңЁ иҝҷдёӨз§ҚзүҲжң¬дёӯзҡ„е…·дҪ“е®һзҺ°е…ізі»еӣҫгҖӮ

жҠҪиұЎ ActionBar.Tab

еҲӣе»әжӮЁзҡ„жҠҪиұЎtabзұ»пјҢиҜҘзұ»зҡ„еҮҪж•°еҹәжң¬е’ҢВ [ActionBar.Tab](http://developer.android.com/reference/android/app/ActionBar.Tab.html)В жҺҘеҸЈдёҖж ·:

public abstract class CompatTab {

иҝҷйҮҢдҪҝз”ЁжҠҪиұЎзұ»иҖҢдёҚжҳҜжҺҘеҸЈжҳҜдёәдәҶз®ҖеҢ–е®һзҺ°дёҖдәӣеёёи§„зҡ„еҠҹиғҪгҖӮе…·дҪ“иҜҰз»Ҷжғ…еҶөеҸӮиҖғзӨәдҫӢйЎ№зӣ®дёӯзҡ„е®Ңж•ҙд»Јз ҒгҖӮ
