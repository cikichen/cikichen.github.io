java split() 分割多字符串的问题

SimonChen
 2013-03-19  27 words   One minute 
Contents

split只能用来分割单字符，比如：

String ip = "172.0.0.1";
//很显然，直接分割
String [] ipArray = ip.split("\.");

但是如果是下面这样的呢？

String ip = "172.,0.,0.,1";
//这样我们可以考虑使用替换，把多字符替换成单字符
String ip1 = ip.replaceAll("\.,", ",");
String [] ipArray = ip1.split(",");
