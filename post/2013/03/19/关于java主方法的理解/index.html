关于java主方法的理解

Written by Simon Chen with ♥ on 19 March 2013 in 1 min

和C语言一样，java的主方法也是main()关键词，而且一个java类中只能有一个主方法，那么有时候看到这样的类

那么这个是正确的吗？答案是肯定的。只有 public static void main(String args[])这样的方法才是主方法，而public void main()这是一个普通的方法，虽然含有main，但是也是合法的。我觉得这是一个容易误解的地方！这实际上是java的重载机制的一种体现！
