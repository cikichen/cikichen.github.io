java.lang.UnsatisfiedLinkError: Could not load SWT library. Reasons

  • UnsatisfiedLinkError

ubuntu12.04安装eclipse后可能出现这个问题，解决办法:

cd ~/.swt/lib/linux/x86
ln -s /usr/lib/jni/* .
