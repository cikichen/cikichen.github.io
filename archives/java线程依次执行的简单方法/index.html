java线程依次执行的简单方法

  • java

有时候需要实现一组线程任务，但是我们希望线程一个执行完成后再去执行另外一个，有一个很简单的方法，就是使用thread的join()方法，如下:

Thread t = new Thread(r);
t.start();
t.join();
