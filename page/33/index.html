Scrollview嵌套HorizontalScrollView导致横向滑动卡顿现象解决

Simon Chen  published on 2013-03-20
也许会有人遇到，在这里说下解决方法。方便以后有人纠结这个问题。开发中经验会遇到滑动里面嵌入滑动的问题，但是这种情况下触摸事件就会发生冲突。导致滑动非常卡，甚至出现程序停止响应。这种情况下我们一般需要重写view。下面给出重新scrollview的方法 public class CustomScrollView extends ScrollView { private GestureDetector mGestureDetector; View.OnTouchListener mGestureListener; @SuppressWarnings("deprecation") public CustomScrollView(Context context,AttributeSet attrs) { super(context,attrs); mGestureDetector= new GestureDetector(new YScrollDetector()); setFadingEdgeLength(0); } @Override public boolean onInterceptTouchEvent(MotionEvent ev) { return super.onInterceptTouchEvent(ev) &amp;&amp; mGestureDetector.onTouchEvent(ev); } // Return false if we're scrolling in the x direction class YScrollDetector extends SimpleOnGestureListener { @Override public boolean onScroll(MotionEvent e1, MotionEvent e2, float distanceX, float distanceY) { if(Math.abs(distanceY) &gt; Math.abs(distanceX)) { return true; } return false; } } } 使用的时候使用这个自定义的控件就可以了。

ubuntu下QQ解决方案

Simon Chen  published on 2013-03-20
ubuntu下QQ是个大问题，试过很多方法都不给力。最后找到一个个人认为是最方便的方案，和大家共享。 必备要求:chrome浏览器 wine+qq方案网上有很多方法，但是能否成功很看人品。所以最后能选择的只有webqq了，但是开着网页用WEBQQ总是不爽的。这里就是用到谷歌浏览器带的一个实用功能:可以把所有网站都创建成一个应用程序的快捷方式。操作方法:打开WEBQQ页面，然后选择谷歌浏览器的文件-创建应用程序快捷方式，确认。桌面上会生成一个快捷图标，可以托到左侧的快速启动栏。如果发现生成的图标不清晰，可以右键-属性，然后点击图标，选择自己喜欢的图片。完成。这样的方式类似于pywebqq,但是pywebqq的桌面提示太差了，延迟很久。不如这个方法实在。
