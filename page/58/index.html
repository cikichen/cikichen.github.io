在日常开发中，你是否遇到过这样的问题：

  • Docker 镜像动辄几个 GB，拉取部署要等好几分钟

什么是单例模式

单例模式（Singleton Pattern）是最常用的设计模式之一，它保证一个类只有一个实例，并提供一个全局访问点。

在实际开发中，单例模式常用于：

  • 数据库连接池
  • 线程池
  • 配置管理器
  • 日志记录器
